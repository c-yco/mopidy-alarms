Mopidy extension for alarms for wake up clock like stuff


Installation
============

Install by running::

    pip install Mopidy-Alarms

Or, if available, install the Debian/Ubuntu package from `apt.mopidy.com
<http://apt.mopidy.com/>`_.


Configuration
=============
